builder class to define the filter cells of a table

Public Instance methods

column(name, opts={})

the method used to actually define the filters of the columns, taking the name of the attribute and a hash of options.

The following options are evaluated here:

:filter_html:a hash with html-attributes added to the <ts>s created
:filter:may take different values:
false:no filter is output for this column
a Hash:the keys of the hash are used to define a select where the values are the value of the options.
an Array:the elements of that array are used to define a select
a subclass of ActiveRecord::Base:a select is created with all instances
