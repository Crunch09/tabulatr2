builder class to define the header cells of a table

Methods

column  

Public Instance methods

column(name, opts={})

the method used to actually define the headers of the columns, taking the name of the attribute and a hash of options.

The following options are evaluated here:

:th_html:a hash with html-attributes added to the <th>s created
:header:if present, the value will be output in the header cell, otherwise, the capitalized name is used
